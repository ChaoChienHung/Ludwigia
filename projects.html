SpaceshipX

SpaceshipX is a small assembly-based game developed using MASM and WinDbg. In the game, the player controls a spaceship to shoot bullets and destroy incoming comets to protect Earth. Future updates will include new items, additional weapon types, and extra lives.

Key Features

  • Control the spaceship using the keyboard: arrow keys (W, S, A, D) to move, and Space to shoot.
  • Shoot bullets to destroy incoming comets
  • Protect the Earth from destruction
  • Planned enhancements: multiple weapon types, collectibles, and advanced enemy patterns

Ludwigia

Ludwigia is my personal website that serves as both a portfolio and a playground for learning essential web development skills. Through this project, I practice core frontend technologies such as HTML, CSS, JavaScript, and Bootstrap while building a space to introduce myself and showcase my work.

Key Features

  • Home page with personal introduction and contact information
  • Projects page featuring a carousel preview of selected work
  • (Optional) Blog section for sharing thoughts and updates
  • Planned enhancements: responsive layout for smartphones and tablets, blogs updates

Mentorion

Mentorion is designed to practice AI function calling while building a learning support application. It is a web application designed to help students learn and test their understanding of the stored notes. The application includes features such as automatic article parsing, note-taking and storage, question generation, and answer evaluation.

Key Features

  • AI Agent Capabilities:
    • Scrape notes from websites (URL input)
    • Generate summary
    • Generate questions & answers
  • Planned enhancements: authentication pages, web interface, dashboard, skill reviewer
